# C/C++ RunAnywhere README

全称 C/C++ Code Run Anywhere

vccode打开c/cpp文件，立即编译并运行， 需要安装gcc。

## Usage

Ctrl + Shift + P 打开命令栏输入： 

run：编译并生成执行文件  
run -s: 编译并生成执行文件（gcc的功能，只限c语言）  
compile -S：编译成汇编代码（拓展名.s，可用文本文件打开）

编译的错误信息可以在控制台查看。

## Requirements

需要安装gcc
